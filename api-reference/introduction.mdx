## Authentication

All API endpoints are authenticated using an API key. You'll need to include your API key in the headers of your requests:

```json
{
  "headers": {
    "X-API-Key": "YOUR_API_KEY"
  }
}
```

<Note>
  Remember to keep your API key secure and never share it publicly. You can manage your API keys in the [Openperplex API Platform](https://api.openperplex.com).
</Note>

## Getting Started

To start using the Openperplex API:

1. Sign up for an account on our [API Platform](https://api.openperplex.com).
2. Obtain your API key from the dashboard.
3. Explore our API endpoints and integrate them into your application.
4. Refer to our [Quickstart Guide](/quickstart) for detailed setup instructions.
